Система футбольной статистики

Football Statistics System (v%%FSS_VERSION%%)
Полностью обособленная динамическая система сбора футбольной статистики в одном файле. Не требует никакого дополнительного софта, кроме браузера (с включенным javascript). Все данные в определенном виде необходимо сохранять самостоятельно простым копи-пастом ;-)

Панель статистики

Это первый запуск приложения. Для начала работы Вам необходимо создать хотя бы один чемпионат и заполнить его командами. Вы можете создать несколько различных турниров с любым количеством команд в каждом.
Для каждой команды можно указать один или несколько статусов, показывающих её заслуги по итогам прошлого сезона, а также цветом выделить зоны еврокубков и вылета.
